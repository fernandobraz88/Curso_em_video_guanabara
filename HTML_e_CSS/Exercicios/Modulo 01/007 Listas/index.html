Trabalhando com Listas

Listas Ordenadas

Listas ordenadas são aquelas que a ordem faz diferença

Para se criar uma lista ordenada, deve-se usar a tag <ol> (ordered list) e cada item da lista deve ser acicionado utilizando a tag <li> (list item)

Ex:
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Trabalhar

Para formatar a lista, na tag de abertura pode ser inserido alguns atrigutos de formatação:

  • type: Modifica o marcador
    • 1 - Numérico (padrão)
    • I - Numérico romano maiusculo
    • i - Numérico romano minusculo
    • A - Alfabetico maiusculo
    • a - Alfabetico minusculo

  • start: indica de qual posição a lista começa, caso ponha em uma lista alfabetica o start= "4" a lista começará a partir da letra "D"
Ex:
  1. Acordar
  2. Tomar café
  3. Escovar os dentes
  4. Trabalhar

no caso o exemplo acima foi formatado com o seguinte padrão: <ol type="A" start="4">

Listas não ordenadas

Listas não ordenadas são aquelas que não possuem ordem de listagem

Para se criar uma lista não ordenada, deve-se usar a tag <ul> (unordered list)

Assim como nas listas ordenadas, os itens da lista serão adicionados utilizando a tag <ui>

Já no que diz respeito a formatação, apenas teremos o atributo type e este modificará apenas o design do marcador

Ex:type=

  1. disc:
    • Pão
    • Manteiga
    • Queijo
  2. square:
    • Pão
    • Manteiga
    • Queijo
  3. circle:
    • Pão
    • Manteiga
    • Queijo

Caso não haja formatação, disc é o padrão
