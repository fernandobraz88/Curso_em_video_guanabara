Principais formatações

Negrito/ Destaque

Nesta frase temos um termo em negrito usando a tag <b> (em desuso pois não possui função semântica)

Nesta frase temos um termo em negrito usando a tag <strong>

Itálico/ Ênfase

Nesta frase temos um termo em italico usando a tag <i> (em desuso pois não possui função semantica)

Nesta frase temos um termo em italico usando a tag <em>

Texto marcado

1.Tambem podemos criar um texto marcado com a tag <mark>. Amarelo é a cor padrão

2.Já neste texto marcado, para alterar a cor, utilizei a css in line style="background-color" dentro da tag de abertura

3.Tambem poderia ter alterado a cor do texto marcado utilizado a tag css style detro do <head>. Porém, para não alterar a cor do <mark> do tópico 1. atribuí a <mark> desse paragrafo a class="mark01"

4. Ainda outra forma de alterar a cor do background, seria fazê-lo diretamente no css3 e fazer o link para este html

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo

Obs: a tag <big> está entrando em desuso pois o tamanho das letras pode ser alterado no css usando a função "font-size". Curiosamente a tag <small> não sofre a mesma condição, talvez seja pela função semântica das letras pequenas em contratos e outros documentos

Texto deletado (riscado)

Podemos marcar um texto como excluído, para enfatisar que ele deve ser lido porém não considerado utilizando a tag <del>

Texto inserido (sublinhado)

Podemos marcar um texto como inserido, para indicar que esse texto foi incluído depois utilizando a tag <ins>

Também existe outra forma de deixar o texto sublinhado que é usando a tag <u>, porém esta, nao tem função semantica e provavemente entrará em deuso

Texto sobrescrito

Texto sobrescrito é aquele que é utilizado por exemplo em exponenciação matemática, como na equação de segundo grau X2+3X+12=0

Para deixar o "2" sobrescrito na equação acima, utilizou-se a tag <sup>

Texto subescrito

Texto subescrito é aquele que é utilizado como por exemplo em formulas quimicas H20(agua), CO2(gás carbonico), C6H12O6(Glicose).

Para deixar os numeros subescritos nas formulas acima, utilizou-se a tag <sub>
